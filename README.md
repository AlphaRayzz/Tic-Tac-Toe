# Tic Tac Toe - OODP Project

## Frontend
Open index.html in any browser to play the game visually.

## Backend (C++)
Compile and run:
  g++ -std=c++17 tictactoe.cpp -o tictactoe
  ./tictactoe

## Concepts Used
- Unit 1: Classes, Constructors, Destructors
- Unit 2: Inheritance, Polymorphism
- Unit 3: Operator Overloading, Friend Functions
- Unit 4: Templates, Exception Handling, File I/O
- Unit 5: STL - vector, array, map, iterators, algorithms
